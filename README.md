# Iván Cumbicus Tacuri — Portfolio (Static)

Sitio estático listo para **GitHub Pages** (sin frameworks, sin build tools).

## Estructura
- `index.html`
- `styles.css`
- `script.js`
- `assets/` (foto, OG image, favicon)

## Publicar en GitHub Pages (repo: `ICumbicus`)
1. Sube estos ficheros a la **raíz** del repositorio `ICumbicus`.
2. En GitHub: **Settings → Pages**
3. **Source**: Deploy from a branch
4. **Branch**: `main` — **Folder**: `/ (root)`
5. Guarda. La web se publicará en: `https://iv4nct.github.io/ICumbicus/`

## CV (opcional)
- Sube tu CV como: `assets/CV_Ivan_Cumbicus.pdf`
- El botón “Descargar CV” se activará automáticamente.
